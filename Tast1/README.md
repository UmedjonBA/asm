# Программа проверки четности чисел на MASM

## Описание проекта

Данный проект содержит три версии программы на ассемблере MASM для проверки четности чисел:

1. **Основная версия** (`even_odd_checker.asm`) - использует функции printf/scanf
2. **Windows API версия** (`even_odd_checker_winapi.asm`) - использует только Windows API
3. **Расширенная версия** (`advanced_number_checker.asm`) - включает проверку простых чисел и обработку нескольких чисел

## Функциональность

### Основные возможности:
- ✅ Ввод целого числа от 1 до 100
- ✅ Проверка четности/нечетности
- ✅ Обработка ошибок ввода
- ✅ Структурированный и прокомментированный код

### Дополнительные возможности (расширенная версия):
- ✅ Проверка простых чисел
- ✅ Обработка нескольких чисел
- ✅ Интерактивное меню
- ✅ Массивы данных

## Сборка и запуск

### Автоматическая сборка (рекомендуется):

1. Откройте командную строку в папке проекта
2. Соберите все версии: `make all`
3. Или соберите конкретную версию: `make even_odd_checker.exe`

### Доступные команды Makefile:

```bash
make all                    # Собрать все версии
make even_odd_checker.exe   # Основная версия
make even_odd_checker_winapi.exe  # Windows API версия
make advanced_number_checker.exe   # Расширенная версия
make clean                  # Удалить временные файлы
make run-basic              # Запустить основную версию
make run-winapi             # Запустить Windows API версию
make run-advanced           # Запустить расширенную версию
make help                   # Показать справку
```

### Ручная сборка (альтернатива):

#### Основная версия:

#### Windows API версия:

```bash
link /subsystem:console even_odd_checker_winapi.obj kernel32.lib user32.lib
```

#### Расширенная версия:

## Использование

### Основная версия:
1. Запустите: `make run-basic`
2. Введите число от 1 до 100
3. Получите результат проверки четности
4. Нажмите Enter для продолжения или 'q' для выхода

### Windows API версия:
- Запустите: `make run-winapi`
- Аналогично основной версии, но использует только Windows API

### Расширенная версия:
1. Запустите: `make run-advanced`
2. Выберите операцию из меню:
   - 1: Проверить четность
   - 2: Проверить простое число
   - 3: Обработать несколько чисел
   - 0: Выход
3. Следуйте инструкциям на экране

## Алгоритмы

### Проверка четности:
```assembly
mov eax, number
and eax, 1      ; Проверяем младший бит
cmp eax, 0
je even_number  ; Если бит = 0, число четное
```

### Проверка простого числа:
```assembly
; Проверяем делимость на числа от 2 до sqrt(number)
mov ecx, 2      ; Делитель
check_divisor:
    mov eax, number
    mov edx, 0
    div ecx
    cmp edx, 0  ; Проверяем остаток
    je not_prime
    inc ecx
    mov eax, ecx
    mul ecx
    cmp eax, number
    jle check_divisor
```

## Обработка ошибок

Программа включает обработку следующих ошибок:
- Некорректный ввод (нечисловые символы)
- Числа вне диапазона (1-100)
- Неверный выбор в меню
