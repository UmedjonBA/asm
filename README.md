# Быстрое создание GUI приложений

Этот репозиторий содержит примеры быстрого создания GUI приложений на разных языках программирования с возможностью сборки в статические исполняемые файлы.

## 🚀 Варианты (от самого быстрого к более сложному)

### 1. Go + Fyne (⭐ РЕКОМЕНДУЕТСЯ)
**Время разработки: 5-10 минут**

#### Преимущества:
- Очень быстрая разработка
- Простой синтаксис
- Кроссплатформенность
- Малый размер exe (~15-20 МБ)
- Не требует установки runtime

#### Сборка:
```bash
# Установка Go (если не установлен)
# Ubuntu/Debian: sudo apt install golang-go
# или скачать с https://golang.org/dl/

# Сборка
./build.sh
```

#### Запуск:
```bash
go run main.go
```

### 2. Rust + egui (современный)
**Время разработки: 10-15 минут**

#### Преимущества:
- Очень быстрое выполнение
- Безопасность памяти
- Современный UI
- Малый размер exe (~5-10 МБ)

#### Сборка:
```bash
cd rust_example
cargo build --release
# Exe файл будет в target/release/
```

### 3. C++ + Qt (мощный)
**Время разработки: 15-30 минут**

#### Преимущества:
- Профессиональный UI
- Огромные возможности
- Нативная производительность

#### Сборка:
```bash
cd qt_example
mkdir build && cd build
cmake ..
make
```

## 🏆 Рекомендации

### Для быстрого прототипирования:
**Go + Fyne** - самый быстрый способ создать рабочее GUI приложение.

### Для серьезных проектов:
**Rust + egui** - современный, быстрый и безопасный.

### Для корпоративных приложений:
**C++ + Qt** - максимум возможностей и профессиональный вид.

## 📦 Сборка статических exe файлов

### Go (Fyne):
```bash
# Windows
CGO_ENABLED=1 GOOS=windows GOARCH=amd64 go build -ldflags "-s -w -H windowsgui" -o app.exe main.go

# Linux
CGO_ENABLED=1 GOOS=linux GOARCH=amd64 go build -ldflags "-s -w" -o app main.go
```

### Rust:
```bash
cargo build --release
# Уже статический по умолчанию
```

### C++ (Qt):
```bash
# Требует статически скомпилированный Qt
cmake -DCMAKE_BUILD_TYPE=Release ..
make
```

## 🎯 Быстрый старт (Go + Fyne)

1. Установите Go
2. Запустите: `./build.sh`
3. Готово! У вас есть `app.exe` для Windows и `app` для Linux

Время от установки до готового exe: **5 минут**!
